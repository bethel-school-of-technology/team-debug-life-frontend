# Setup
### Prerequisites

- NPM (Node Package Manager)
- VS Code

### Instructions

- Clone or download the frontend
- Clone or download the [backend](https://github.com/bethel-school-of-technology/team-debug-life-backend)
- Follow instructions to run the backend
- `npm install`
- `npm start`
